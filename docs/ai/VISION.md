---

## 1. 产品愿景

**PaperGenius Pro** 是一款**集成式 AI 学术写作平台**，采用**多代理系统（Multi-Agent）+ 插件化架构**，实现"一键输入关键词 → 输出可投稿完整论文"的端到端自动化工作流。

### 核心价值主张
- **效率提升 10 倍**：从选题到投稿平均耗时从 3 个月缩短至 3 天
- **学术严谨性**：所有生成内容强制带真实引用 + RAG 溯源 + 逻辑自洽检查
- **隐私与灵活**：本地 Docker 一键部署（Ollama 全离线）或云端 API
- **插件化**：任何 GitHub 项目可作为插件接入

### 产品定位
Notion + Overleaf + Zotero + Elicit + STORM 的超级融合体，专为中国高校/科研院所 + 全球研究者设计。
